#Experience Generator

Generate "realistically looking" traffic for Sitecore sites with configurable patterns, including:

 - Trends over time
 - Identified contacts with multiple visits
 - Bounce rate
 - Geo location
 - Landing pages
 - Channels
 - Referrers
 - Internal and external search
 - Outcomes
 - Campaigns
